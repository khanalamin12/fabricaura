Change Admin Password

          
Your password lives in Netlify, not in this site's code

        

        

          For security, the admin password is stored as an environment variable on Netlify —
          it's never saved in your website files. To change it:
        

        
    
          
  1. Open your Netlify dashboard and select this site.
    2. 
          
  2. Go to Site configuration → Environment variables.
    3. 
          
  3. Find ADMIN_PASSWORD, click it, and edit the value.
    4. 
          
  4. Save, then trigger a redeploy (Deploys → Trigger deploy → Deploy site) if it doesn't redeploy automatically.
    5. 
        

        

          This step has to be done on Netlify directly — letting the admin panel itself change the
          password would mean giving it the power to edit your site's settings, which is riskier
          than it's worth. Pick something strong; this password protects your entire product catalog.
